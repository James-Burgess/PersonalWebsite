Here are some things I've built. More on [GitHub](https://github.com/James-Burgess?tab=repositories).

> [Field Log](https://thefieldco.com/field-log) · active
>
> Offline-first research logger for conservation fieldwork. Built because fieldwork happens where there's no signal. IndexedDB, camera integration, GPS tagging. Runs for weeks without a network, syncs when one appears. Working on getting it into real hands.

> CloudIntelligence.africa · 2020
>
> Walked into fourteen MSSQL servers, each with its own schema and a week-long ETL process. The senior DBA had just quit. Built a custom migration pipeline (Python, SQLAlchemy, Alembic, PostgreSQL, pandas), unified fourteen schemas into one, and moved it live in production. One week per run became one minute. The system has run untouched for years—the only updates have been adding new clients.

> [whereami](https://github.com/James-Burgess/whereami) · iot
>
> Live satellite tracker via SPOT beacon with HTML5 map rendering. Built to track long-distance sailing passages and remote fieldwork. The internet made the hardware obsolete but the code still works.
